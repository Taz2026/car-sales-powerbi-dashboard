# Car Sales Power BI Dashboard

## Project Overview
This dashboard provides a comprehensive analysis of vehicle sales data, focusing on revenue trends, quantity by segment, and city-level performance.

## Key Features
* **Sales Trends:** Visualization of profit vs. quantity across different segments.
* **Geographic Analysis:** Revenue breakdown by city.
* **DAX Measures:** Custom calculations for key business metrics.

## Files in this Repository
* **SalesDashboard.pbix:** The main Power BI project file.
* **dataTopCars_Africa.xlsx:** The source dataset used for analysis.
